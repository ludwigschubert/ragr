Overview

RKAbstractTableController is an abstract base class for concrete table controller classes. A table controller object acts as both the delegate and data source for a UITableView object and leverages the RestKit object mapping engine to transform local domain models into UITableViewCell representations. Concrete implementations are provided for the display of static table views and Core Data backed fetched results controller basied table views.
